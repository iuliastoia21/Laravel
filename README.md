# Women in FinTech

Women in FinTech is a web application developed with Laravel, designed to support the management of a professional community focused on women working in or interested in the fintech industry.

The application allows the management of community members by storing information such as name, email, profession, company, LinkedIn profile and status. It also includes features for adding and viewing success stories associated with members, as well as creating and managing community events.

The project uses Laravel’s MVC architecture, Eloquent ORM for database interaction, Blade templates for the interface, and Bootstrap for responsive styling. The application also includes validation rules, pagination, search and filtering options, and CSV export for member data.

## Main Features

* Member management: create, edit, delete and view members
* Search and filtering by name, email, profession, company and status
* Success stories linked to specific members
* Event management with event date and description
* CSV export for member records
* Data validation for forms
* Pagination for easier navigation
* Responsive interface built with Bootstrap

## Technologies Used
